# AI-Based Dyslexia Detection

This project detects dyslexia in handwritten text using deep learning and OCR.

## Features
- CNN model for character-level detection
- Flask web app for analysis
- OpenCV + Tesseract OCR for preprocessing

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
